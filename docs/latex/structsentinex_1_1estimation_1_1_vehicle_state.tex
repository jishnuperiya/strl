\doxysubsection{Detailed Description}
Represents the state of a vehicle for state estimation.

The state is expressed in a global/\+world reference frame and follows a fixed ordering which must be consistent with the associated covariance matrix.

State vector:\+   \[\mathbf{x} = [x, y, \psi, v]^T
\]

Units:\+
\begin{DoxyItemize}
\item x, y :\+ meters
\item psi :\+ radians
\item v :\+ meters per second 
\end{DoxyItemize}
