\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{index_intro_sec}
{\bfseries{Sentinex}} is a modern C++20 sensor fusion and simulation suite designed for robust vehicle state estimation and system-\/level validation.

The project focuses on deterministic, testable, and extensible estimation pipelines suitable for automotive, robotics, and simulation-\/driven development workflows.\hypertarget{index_goals_sec}{}\doxysection{\texorpdfstring{Design Goals}{Design Goals}}\label{index_goals_sec}

\begin{DoxyItemize}
\item Strongly typed state and measurement representations
\item Clear separation between models, filters, and data sources
\item Deterministic simulation and replay of sensor data
\item Emphasis on correctness, observability, and numerical stability
\item Modern C++20 design with minimal runtime overhead
\end{DoxyItemize}\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Core Features}{Core Features}}\label{index_features_sec}

\begin{DoxyItemize}
\item Vehicle state estimation using Kalman filtering techniques (linear KF, EKF, extensible to UKF)
\item Explicit process and measurement models
\item Sensor abstraction for simulation and replay
\item System-\/level validation utilities
\item Well-\/documented APIs using Doxygen
\end{DoxyItemize}\hypertarget{index_architecture_sec}{}\doxysection{\texorpdfstring{Architecture}{Architecture}}\label{index_architecture_sec}
Sentinex is structured around a small set of orthogonal components:\+


\begin{DoxyItemize}
\item {\bfseries{State representations}} (e.\+g., Vehicle\+State)
\item {\bfseries{Dynamic models}} (process and motion models)
\item {\bfseries{Measurement models}} (sensor-\/specific observation models)
\item {\bfseries{Filters}} (Kalman Filter variants)
\item {\bfseries{Simulation and validation tools}}
\end{DoxyItemize}

Headers define the public API and semantic contracts, while implementation files focus on numerical methods and algorithms.\hypertarget{index_standards_sec}{}\doxysection{\texorpdfstring{Coding Standards}{Coding Standards}}\label{index_standards_sec}

\begin{DoxyItemize}
\item C++20
\item Header-\/only documentation using Doxygen
\item No hidden ownership or implicit state
\item Explicit units and coordinate frame conventions
\end{DoxyItemize}\hypertarget{index_audience_sec}{}\doxysection{\texorpdfstring{Intended Audience}{Intended Audience}}\label{index_audience_sec}
Sentinex is intended for:\+
\begin{DoxyItemize}
\item Engineers working on vehicle state estimation
\item Simulation and validation engineers
\item Researchers and students studying sensor fusion
\item Developers interested in modern C++ estimation architectures 
\end{DoxyItemize}
